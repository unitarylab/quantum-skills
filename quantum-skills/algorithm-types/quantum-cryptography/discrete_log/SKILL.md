# Shor's Discrete Logarithm Algorithm - Hidden Order Problem Skill

## Overview
**Shor's Discrete Logarithm Algorithm** is a quantum algorithm that solves the discrete logarithm problem in polynomial time. Given a prime modulus P, generator g, and element y = g^x mod P, the algorithm finds the exponent x exponentially faster than any known classical algorithm.

### Why Discrete Logarithm Algorithm Matters
- **Cryptographic Impact**: Breaks Diffie-Hellman key exchange and elliptic curve cryptography
- **Complexity**: O(n³) quantum time vs sub-exponential classical time (exponential speedup)
- **Foundational**: Demonstrates quantum advantage in number-theoretic problems
- **Real-world Threat**: Endangers current internet security infrastructure

## Mathematical Background

### The Discrete Logarithm Problem (DLP)
- **Input**: Prime P, generator g ∈ ℤₚ*, element y ∈ ℤₚ*
- **Promise**: ∃x ∈ {0,1,...,P-2} such that g^x ≡ y mod P
- **Goal**: Find the smallest non-negative x satisfying the equation
- **Hardness**: Classical algorithms require sub-exponential time O(exp(c√(log P log log P)))

### Hidden Order in Abelian Groups
The discrete log problem can be viewed as finding the order of y in the cyclic subgroup generated by g. The quantum algorithm exploits the periodic structure in the multiplicative group ℤₚ*.

### Key Quantum Concepts

**1. Quantum Fourier Transform over Finite Fields**
- Classical FFT: Transforms between time/frequency domains
- Quantum QFT: Transforms between computational/Hadamard bases
- Application: Extracts periodic information from quantum superpositions

**2. Phase Estimation**
```
For unitary U with eigenvalue e^(2πiθ), phase estimation finds θ
In DLP: U_g |z⟩ = |g·z mod P⟩ has eigenvalues e^(2πi·s/r)
Result: Measurement reveals fractional parts of s/r
```

**3. Continued Fractions**
- Classical tool for extracting rational approximations
- Quantum measurement gives u/N ≈ s/r
- Continued fraction expansion recovers r and s

## Algorithm Structure

### The Five-Phase Shor's Discrete Log Algorithm

#### Phase 1: Parameter Setup
```
1. Validate inputs: gcd(g,P)=1, gcd(y,P)=1
2. Set register sizes: n_count = 2*log₂(P), n_work = log₂(P)
3. Initialize quantum registers: A (n_count), B (n_count), Work (n_work)
```

#### Phase 2: Quantum Circuit Construction
```
1. Initialize superposition: H^⊗(2n) on registers A,B
2. Initialize work register: |work⟩ = |1⟩
3. Apply controlled modular multiplication: U_g^a ⊗ U_{y⁻¹}^b
4. Apply inverse QFT: IQFT on register A, IQFT on register B
```

#### Phase 3: Quantum Execution
```
1. Execute quantum circuit
2. Measure probability distribution over A,B registers
3. Extract high-probability measurement outcomes
```

#### Phase 4: Classical Post-Processing
```
1. Continued fraction expansion: Extract r,s from u/N ≈ s/r
2. Linear congruence solving: Solve s·x ≡ -round(v·r/N) mod r
3. Verification: Check if g^x ≡ y mod P
```

#### Phase 5: Result Validation and Output
```
1. Verify solution satisfies original equation
2. Generate circuit diagram and timing statistics
3. Format results for analysis
```

```python
def mod_inverse(a, m):
    """Compute modular inverse using extended Euclidean algorithm"""
    m0, y, x = m, 0, 1
    if m == 1:
        return 0
    while a > 1:
        q = a // m
        m, a = a % m, m
        y, x = x - q * y, y
    if x < 0:
        x += m0
    return x

def mod_pow(base, exp, mod):
    """Modular exponentiation"""
    result = 1
    base = base % mod
    while exp > 0:
        if exp % 2 == 1:
            result = (result * base) % mod
        exp = exp >> 1
        base = (base * base) % mod
    return result
```

#### 2. Continued Fraction Expansion
```python
def continued_fractions(a, b, max_terms=20):
    """Compute continued fraction expansion of a/b"""
    cf = []
    for _ in range(max_terms):
        cf.append(a // b)
        a, b = b, a % b
        if b == 0:
            break
    return cf

def convergents(cf):
    """Compute convergents from continued fraction"""
    h, k = [0, 1], [1, 0]
    for q in cf:
        h.append(q * h[-1] + h[-2])
        k.append(q * k[-1] + k[-2])
    return list(zip(h[1:], k[1:]))
```

## Mathematical Deep Dive

### Why Quantum Fourier Transform Works
```
The quantum Fourier transform extracts phase information:
QFT |k⟩ = (1/√N) Σ_{j=0}^{N-1} ω^{jk} |j⟩ where ω = e^{2πi/N}

For periodic functions f(x) = f(x+r), the QFT reveals the period
through constructive/destructive interference in phase space.
```

### Continued Fractions and Diophantine Approximation
```
The key insight: Quantum measurement gives u/N ≈ s/r
Continued fraction expansion finds best rational approximations:

u/N = s/r + ε, where |ε| < 1/(2N)

The convergents h_k/k_k satisfy |u/N - h_k/k_k| < 1/(k_k²)
Thus we can recover r and s from the measurement.
```

### Linear Congruence Solving
```
From measurements: u/N ≈ s/r, v/N ≈ -s·x/r

We get: s·x ≡ -round(v·r/N) mod r

This is solved using the extended Euclidean algorithm,
finding x in the coset of solutions modulo the order.
```
